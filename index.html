1. Write a code to insert an image

The <img> tag is used to embed an image in an HTML page.


The <img> tag has two required attributes:

    .src - Specifies the path to the image

    .alt - Specifies an alternate text for the image, if the image for some reason cannot be displayed


syntax of image tag :-    <img src="path to the image" alt="#">

3. Creat a table and explain how to achive colspan

the colspan attribute defines the number of columns a cell should merge.

        code of table with colspan :-

        <table border="1">
            <tr>
              <th>Day</th>
              <th>Savings</th>
            </tr>
            <tr>
              <td>Monday</td>
              <td>Rs100</td>
            </tr>
            <tr>
              <td>Tuesday</td>
              <td>Rs80</td>
            </tr>
            <tr>
              <td colspan="2">Sum: Rs180</td>here two column merge
            </tr>
          </table>
Output:-

Day Savings
Monday Rs100
Tuesday Rs80
Sum: Rs180
